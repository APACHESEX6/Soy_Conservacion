# Scripts del Proyecto

Esta carpeta contiene scripts utilitarios para el monorepo.

## Scripts Disponibles

- `setup.sh` - Configuración inicial del entorno
- `clean.sh` - Limpieza de cache y archivos temporales
- `deploy.sh` - Script de despliegue

## Uso

Ejecuta los scripts desde la raíz del proyecto:

```bash
pnpm run setup
pnpm run clean
```
